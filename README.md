# unex-fusion-sim

This project provides simple Streamlit apps for exploring fusion concepts.

* `app.py` contains the original "Live Smart Mode" tuner.
* `real_model_app.py` implements a minimal D-T fusion model using
  basic physics formulas. Run it locally with:
  
  ```bash
  streamlit run real_model_app.py
  ```

These apps are purely educational and should not be taken as accurate
simulations of an actual reactor.
